How is the staging area different from the working directory and the repository? What value do you think it offers?

The staging area is things from the working directory that I want to commit to the repository. It allows multiples files to be grouped into 1 commit without forcing me to commit all changes.


How can you use the staging area to make sure you have one commit per logical change?
Only stage changes that should go together. Commit that set of changes, then stage the next set of changes.
